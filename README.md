Customer Sales Report Dashboard (Excel)


Project Overview
This project is an interactive Customer Sales Report Dashboard built entirely in Microsoft Excel. It analyzes customer and call centre data to provide meaningful business insights through dynamic reports and KPI metrics.

The dashboard enables users to monitor sales performance, customer satisfaction, call statistics, and representative-wise performance using Excel's analytical capabilities.

Features
Interactive Excel Dashboard
Dynamic KPI Cards
Total Call Count
Total Amount
Happy Customers Count
Average Customer Rating
Total Call Duration
Representative-wise (R01, R02, etc.) analysis
Customer and Call Centre data reporting
Pivot Table-based data analysis
Dynamic formulas using INDEX-MATCH and named ranges
Slicers for easy filtering and navigation
Automated report generation based on selected filters

Project Outcome
This dashboard demonstrates how Excel can be used as a powerful business intelligence tool for data analysis and reporting. It provides an easy-to-use interface for stakeholders to explore customer and sales performance without requiring advanced BI tools.
